## What the talk adds beyond the blog series

Based on the searchable abstract and slides preview:

- **Genre framing** — idle games as a real game-design category, not "joke games"
- **Monetization angle** — pacing the prestige loop so it aligns with ad-watch / IAP windows
- **Onboarding research from Kongregate** — first-session retention is driven by visible production rates inside the first 60 seconds
- **Production-cost gap diagrams** — visual restatement of the principle "exponential cost / linear (or polynomial) production creates the gap that paces play"
- **Generator-relevance visuals** — chart of "owned generators by tier over time" showing the collapse to newest-only without intervention
